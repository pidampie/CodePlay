User Manual

01

Akses Sistem

Klik tombol 'Play' pada halaman utama untuk memecahkan sistem keamanan dan masuk ke Navigation Hub.

→
02

Eksplorasi Materi

Buka menu 'Materi Pembelajaran' untuk membaca dan memahami landasan teori dasar serta struktur algoritma pemrograman.

→
03

Tonton Video Pembelajaran

Akses menu 'Video Pembelajaran' untuk menyaksikan penjelasan visual serta contoh demo implementasi penulisan kode secara komprehensif.

→
04

Evaluasi Mandiri

Setelah menguasai materi teori dan visual, uji ketajaman logika berpikir Anda melalui modul evaluasi interaktif pada menu 'Game'.

→
05

Konfigurasi

Sesuaikan preferensi kenyamanan belajar Anda melalui menu 'Pengaturan' untuk mengatur kontrol volume suara latar belakang (BGM).
